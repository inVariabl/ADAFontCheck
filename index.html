ADA Font Check

Determines if a font meets the ADA Accessiblity Guidelines (ADAAG) for tactile signage using opentype.js.

Tests

  1. Character Stroke Width ratio of 10%–20% using the uppercase letters “I,” “H,” and “O”.
  2. Character Body Width ratio of 60%–100% using the uppercase letters “I,” “H,” and “O”.
  3. Italic Font detection.
  4. Serif Font detection.
  5. Sans Serif Font determination.
How To Use
  1. Select “All Fonts” to display both Pass and Fail fonts. Select “ADA Fonts Only” to display Pass fonts.
  2. Select “Folder Upload” button for multiple fonts or “File Upload” button for a single font.
  3. Navigate to the source file in your fonts library.
  4. Select the fonts.
  5. Review the test results by clicking on the font name. Note that results appear for Sans Serif fonts only. Fonts that are not Sans Serif do not comply with ADA requirements.
Notes
  1. Fonts must meet the ratio requirements of both Tests 1 and 2 for ADA compliance.
  2. ADA compliance is based on an overall average of uppercase letter width to height.
  3. Numerals are not tested for compliance but are considered compatible when used with letters of the same font set and size.
